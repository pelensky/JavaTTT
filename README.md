## Java Tic Tac Toe 

## Instructions
1. Clone the repository
2. CD into the repository
3. Build the app by running `$ gradle build`
4. Run the tests by running `$ gradle test`
4. Run the app by running `$ gradle --console plain run`
5. Enjoy!
